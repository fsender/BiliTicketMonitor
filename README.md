## 特点:

1. 免登录, 双击即用, 实时监测B站项目票务信息. Login is not necessary!

2. 自定义脚本, 发现有票时就自动启动脚本, 可关联抢票辅助软件, 或者鼠标宏, 模拟屏幕点击脚本, 相比API请求, 更接近手动抢票, 不易风控.

3. 自由度高, 可以自己设置脚本启动批处理文件, 也可以自己设置请求API!
